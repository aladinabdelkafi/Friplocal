Pour commencer:
	telecharger le projet
	telecharger la base de données "frip.sql"

Pré-requis
	telecharger et installer un serveur Web local par exemple :XAMPP

Démarrage
	placer le projet dans les dossier "dashboard" serveur Web local XAMPP

	activer le service MySql et importer la base de données "frip.sql"

	activer le service Apache et ouvrire le lien  "http://localhost/dashboard/"
